WaRee11

River Simulation for iPad

A mobile hydraulic modeling application designed to simulate 1D river flow using the Saint-Venant equations directly on iPadOS.

WaRee11 aims to provide a portable alternative to traditional desktop hydraulic modeling software, enabling engineers, researchers, and students to build and run river simulations anywhere.


Overview

WaRee11 is a hydraulic simulation platform developed using Swift and UIKit for iPad.

The application allows users to:
	•	Build river networks
	•	Define cross-section geometries
	•	Generate rating curves
	•	Configure simulation settings
	•	Run hydraulic simulations
	•	Visualize water level and discharge results

The project is inspired by professional tools such as:
	•	MIKE 11
	•	HEC-RAS

but designed specifically for mobile devices and simplified workflows.
